DATA TYPES

In programming,data type is an imprtant concept.Variables can store data of different types and different types can do different things.Python has the following data types built-in by default,in these categories:

Text type str
Numeric Types int,float,complex
Sequence Types list,tuple,range
Mapping type dict
Boolean Type bool
Set types and Binary types set,bytes

Getting the data type:

You can get the data type of any object by using the type() function:

Try it
      
        x=5
        print(type(x))
